Odoo Advance Signup

Odoo Advance Signup is a module which helps to build customized sign up page for the Odoo Website. Admin can configure fields for the signup page as per their requirement. This module provides well-designed Odoo login/signup/ page.

This module has been made Multi Website Compatible for the ease of users.

Highlighted Features

  • The module provides well-designed Login/Signup/Reset Password Page.
  • The module is Multi-Website Compatible.
  • Admin can configure their own customized signup page.
  • On successful signup, all the signup values will be stored in the logged in user.
  • Admin can set dynamic fields in the signup form according to their requirement.
  • Admin can also add the dynamic login/signup/reset password content from the backend.
  • Option to hide/show the header/footer from the signup page.
  • This module is Multi-Language compatible.
  • Admin can set a background image for the login/signup page as per their need.
